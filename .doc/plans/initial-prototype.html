How to set up Notion
▾
1
Go to notion.so/my-integrations and click New integration. Give it a name like "Loopd". Copy the integration token.
2
Create a Notion database with all the required properties (see the build plan). Or duplicate the Loopd template if you have a link.
3
Open your database → click Share in the top right → search for your integration name → click Invite.
4
Copy the database ID from the URL. It looks like: notion.so/workspace/DATABASE_ID?v=... — the part between the last / and the ?.
5
Paste both values above and tap Test connection. A green status means you're connected.
